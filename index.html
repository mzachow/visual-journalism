Running 2020

Since March 2020 I have been recording my running activities with a Garmin Vivoactive 4S. As a Data Scientist, I have been looking forward for a long time to analyze all my running data to find interesting insights and to make use of them to improve my training plan. I started to learn from scratch how to develope interactive data visualizations using javascript's library D3. Special thanks to Benjamin Schumann who introduced me to it.
